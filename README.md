# \# Python TCP Port Scanner

# 

# \## Overview

# 

# This project is a lightweight TCP Connect Port Scanner built in Python.  

# It scans a target host over a specified port range and identifies open TCP ports.

# 

# The scanner uses Pythonâ€™s built-in `socket` module to establish TCP connections and determine port status.

# 

# This tool was developed for educational purposes and authorized security testing only.

# 

# ---

# 

# \## Features

# 

# \- TCP Connect scanning

# \- Custom port range selection

# \- Hostname to IP resolution

# \- Timeout handling

# \- Clean terminal output

# \- Error handling

# 

# ---

# 

# \## Installation

# 

# Clone the repository:

# 

# git clone https://github.com/Kasolia/python-tcp-port-scanner.git

# cd python-tcp-port-scanner

# 

# Ensure Python 3 is installed:

# 

# python --version

# 

# ---

# 

### Usage

Run the scanner:

python port_scanner.py <target> <start_port> <end_port>

Example:

python port_scanner.py scanme.nmap.org 20 100
